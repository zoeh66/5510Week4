BBC News Classification Mini-Project

This project explores the BBC News article data from the Kaggle competition available here: https://www.kaggle.com/competitions/learn-ai-bbc/submissions. The goal is to predict the test dataset with teh highest accuracy. This project primarily uses the unsupervised learning technique non-negative matrix factorization (NMF) to classify the articles into their respective categories: business, tech, politics, sport, and entertainment. I also compare the results of my refined NMF model to a supervised learning method, multi-class binomial regression.

Natural language processing with TF-IDF

As shown in the above word cloud, while there are some frequent words in an article that are unique to the article or category, there are also some common words that will likely be frequent in any given article. One example is the word "said", which will be popular across categories of news articles because of the frequent inclusion of interview quotations. To combat this potential problem, I use the Term Frequency-Inverse Document Frequency (TF-IDF) method. This method converts strings into a matrix, where each value is the frequency of a given word in the string normalized by the word's popularity across all strings. Because of this calculation, and the inclusion of the max_df parameter in the function below, very common words that could clutter the model are removed, and others are given a lower value to reflect their lower importance to the model.

Unsupervised learning model

For the unsupervised learning model, I am using sklearn's non-negative matrix factorization (NMF) method. The number of components is set to 5 because there are 5 categories of news articles in the dataset. I begin by creating a few helper functions to automate to process of assigning the categories based on the model's label results, predicting categories for new labels, and saving the test dataset results for submission to the Kaggle competition.

Helper functions

In[9]:
# Helper function to assign categories based on highest accuracy of predictions
def permute_label(df, pred):
    """
    df: training dataset
    pred: model prediction output
    returns best label permutation and accuracy score
    """
    # Initialize values
    best_label = None
    best_acc = 0
    df_labels = list(df.Category.unique())

    # Iterate through all permutations of category combinations
    for perm in itertools.permutations(df_labels):
        pred_map = [perm[i] for i in pred]
        acc = accuracy_score(df['Category'], pred_map)

        if acc > best_acc:
            best_label = [(df_labels.index(i), i) for i in perm]
            best_acc = acc

    # Return results
    return best_label, best_acc
In[10]:
 # Helper function to predict labels for dataset
def predict_label(model, df, best_label):
    """
    model: trained nmf model
    df: test dataset
    best_label: best label permutation from permute_label function
    returns list of predicted labels
    """
    # Transform dataframe with TF-IDF
    df_text = tfidf.transform(df['Text'])

    # Predict values based on model and assign best labels
    pred = model.transform(df_text).argmax(axis=1)
    pred_map = [best_label[i][1] for i in pred]

    # Return results
    return pred_map

# Helper function to save test predictions for submission to the competition
def save_predict(df, pred_map):
    """
    df: test dataset
    pred_map: list of predicted labels
    saves dataframe of predicted labels as csv for submission
    """
    df_pred = pd.DataFrame(columns=['ArticleId', 'Category'])
    df_pred['ArticleId'] = df['ArticleId']
    df_pred['Category'] = pred_map
    df_pred.to_csv('test_predictions.csv', index=False)

Discussion

For the unsupervised learning method, I found that non-negative matrix factorization (NMF) worked best with the 'nndsvda' initializer, the 'mu' (multiplicative update) solver, and the 'kullback-leibler' beta loss parameter. This model worked best in combination with the TF-IDF text processing method with a minimum df of 0.9, a maximum df of 2, and an ngram range of 1 to 3. This final model achieved 94.86% accuracy on the training dataset and 96.33% accuracy on the test dataset. While this model could likely be improved with other refinement techniques or unsupervised learning methods, as-is it was outperformed by the supervised learning technique I chose, multi-class logistic regression. This model was slightly slower than the NMF model to train, but it also led to a 98.37% accuracy on the test data set, 2 full percentage points higher than the NMF results. I did try reducing the training dataset size to make it more efficient, but this quickly led to overfitting and a sharp reduction in accuracy for the test dataset.
